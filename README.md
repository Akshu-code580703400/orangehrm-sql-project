# OrangeHRM SQL Database Project

This project simulates a Human Resource Management System database inspired by OrangeHRM.  
It demonstrates SQL database design, data insertion, and analytical queries for HR operations.

## Tools Used
- MySQL

## Database Tables
- Employees
- Departments
- Leave Requests
- Attendance
- Recruitment
- Performance Reviews
- Payroll

## SQL Concepts Used
- SELECT queries
- JOIN operations
- GROUP BY
- HAVING
- Aggregate functions (COUNT, SUM, AVG)
- Subqueries / Nested queries

## Project Files
schema.sql – Database and table creation  
insert_data.sql – Sample data insertion  
queries.sql – SQL queries for data analysis  

## Sample Queries
- Employee details with department names
- Employees earning above average salary
- Employee count by department
- Leave requests by employees
- Attendance records for employees
- Payroll and salary analysis

## Purpose
This project demonstrates practical SQL skills for database management and HR data analysis.
